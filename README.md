# Prodesk IT Landing Page

A modern, responsive landing page developed as part of Sprint 01 at Prodesk IT. The project demonstrates frontend fundamentals including responsive design, Flexbox, CSS Grid, JavaScript interactivity, and UI/UX best practices.

## Project Overview

This landing page was built for the digital marketing wing of Prodesk IT. The objective was to create a professional, mobile-friendly landing page while adhering to raw HTML, CSS, and JavaScript requirements.

## Features

* Responsive Navigation Bar
* Mobile Hamburger Menu
* Hero Section with Call-to-Action
* Services Section using CSS Grid
* Dark / Light Mode Toggle
* Sticky Navigation Bar
* Hover Animations and Micro-Interactions
* Social Media Footer Icons
* Professional SaaS-inspired UI Design
* Fully Responsive for Mobile and Desktop Devices

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (ES6)
* Font Awesome Icons
