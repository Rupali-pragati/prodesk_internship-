# Digital Movie Database Search

## Overview

Digital Movie Database Search is a React and TypeScript-based web application that enables users to search for movies using an online movie database API. The application provides a clean, responsive interface with proper loading states, error handling, accessibility support, and secure input sanitization.

---

## Features

- Search movies by title
- Real-time movie search using native Fetch API
- Loading indicator during API requests
- User-friendly empty state for no search results
- Error handling for network and server failures
- Input validation and sanitization
- Responsive design for desktop and mobile devices
- Accessible interface with ARIA support
- Telemetry simulation using console logging

---

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Native Fetch API

---

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd digital-movie-database-search
```

Install the required dependencies:

bash
npm install
```

---

## Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## Building for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```

---

## TypeScript Compilation
