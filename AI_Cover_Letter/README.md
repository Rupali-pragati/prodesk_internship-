# 📄 AI Cover Letter Generator

An intelligent, fast, and highly customizable cover letter generator powered by AI. This tool helps job seekers create tailored, ATS-friendly cover letters in seconds by analyzing their resume and the target job description. 

Built with React 19, Vite, and Google Gemini Flash.

## 🚀 Getting Started

Follow these instructions to set up the project and run it locally on your machine.

### Prerequisites

* [Node.js](https://nodejs.org/) (v18 or higher recommended)
* A Google Gemini API key (Get one from [Google AI Studio](https://aistudio.google.com/))

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/ai-cover-letter-generator.git
   cd ai-cover-letter-generator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the local URL provided (usually `http://localhost:5173`).
